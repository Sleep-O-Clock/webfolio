# Mon parcours

Passionné par le développement logiciel depuis plusieurs années, j'ai acquis une solide expérience dans la création d'applications web modernes et évolutives. Mon parcours m'a permis de travailler sur divers projets, du développement full-stack à l'architecture de systèmes complexes.

## Ce qui me motive

Je me concentre sur :
- La création de produits de haute qualité avec une architecture propre
- La création d'excellentes expériences développeur (DX)
- L'écriture de code maintenable et évolutif
- L'apprentissage et l'amélioration continus

## Ce que j'aime construire

- **APIs avec NestJS / PostgreSQL** - Systèmes backend robustes avec typage fort
- **Fonctionnalités en temps réel et messagerie** - Expériences interactives pour les utilisateurs
- **Modèles de données propres et migrations** - Schémas de base de données bien conçus
- **UIs rapides avec Next.js** - Applications frontend modernes et performantes
